# RevClerx (v2)

Editorial, foaster-inspired marketing site for RevClerx. Built with Next.js 14, Tailwind CSS, and a small in-house component layer.

## Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3.4
- TypeScript 5
- framer-motion (light usage)

## Routes

- `/` Home
- `/solutions`
- `/how-it-works`
- `/about`
- `/contact`

## Local development

```bash
npm install
npm run dev
```

Dev server runs on port 3001 to avoid colliding with v1.

## Notes

V2 is intentionally minimal. The visual language is editorial: warm cream surface, dark ink text, single warm-orange accent used sparingly. No heavy effects, no gradient text, no decorative components beyond what is in `/components`.
